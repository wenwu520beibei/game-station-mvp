# 🚀 Cloudflare部署指南

## 你需要做的（2分钟）

1. 打开 https://dash.cloudflare.com
2. 登录账号
3. 点击左侧 **Pages**
4. 点击 **Create project**
5. 选择 **Connect to Git**
6. 选择GitHub，授权（如果需要）
7. 选择仓库 `wenwu520beibei/game-station-mvp`
8. 点击 **Begin setup**
9. 配置：
   - Project name: `game-station-mvp`
   - Production branch: `main`
   - Build command: (留空)
   - Build output directory: (留空)
10. 点击 **Save and Deploy**

## 等待1-2分钟，部署完成！

## 访问地址
部署完成后，你会得到一个类似这样的地址：
